# ERPT_ML_Research

## Project Overview

This repository contains the implementation of my Honours research project, which investigates the use of econometric and machine learning techniques to model and forecast ERP-related outcomes.

The project follows a complete end-to-end data analytics workflow, beginning with raw data collection and ending with model evaluation and interpretation.

---

## Research Pipeline

- Project Setup
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Stationarity Testing
- Econometric Modelling
- Machine Learning Modelling
- Model Evaluation
- Results and Interpretation
